Detailed Description

Cleans the request body so, that only the body of the request is left in the string.

This is needed when calling the endpoint that consumes MediaType.APPLICATION_FORM_URLENCODED_VALUE The actual body needed is stored as a key, and not a value for a key Example of body sent as a parameter: {channel=[CHANNEL_11111], {"sourcetype": "mysourcetype", "event": "Hello, world!"}=[]}
